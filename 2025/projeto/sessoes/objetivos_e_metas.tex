\section{Breve descrição dos objetivos e metas}
\label{sec:objetivos}

Este projeto tem o objetivo de \textbf{avaliar diferentes técnicas de extração de características para o reconhecimento de usuários pela impressão digital}. Para isso, será estudado o processo de extração de características a partir de imagens de impressão digital e sua representação. Algumas técnicas que podem ser consideradas nessa avaliação são as descritas em~\cite{TangMi2},~\cite{TangMi1} e~\cite{Jin2016}.

Os objetivos específicos do projeto são:

\begin{itemize}
    \item Selecionar conjuntos de dados de impressão digital (ver Seção~\ref{sec:metodologia});
    \item Selecionar técnicas de extração de características para reconhecimento de usuários pela impressão digital;
    \item Realizar experimentos comparando as diferentes técnicas de extração de características;
    \item Avaliar as técnicas de extração de características sob diversos aspectos.
\end{itemize}
