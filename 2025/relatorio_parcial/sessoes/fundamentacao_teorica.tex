\section{Fundamentação teórica}\label{sec:fundamentacao_teorica}

Esta seção apresenta algumas constatações sobre a pesquisa na área de impressões digitais e técnicas de extração de características. Com o objetivo de selecionar os métodos que seriam avaliados neste projeto foi feita uma revisão bibliográfica sobre artigos que descreviam métodos de extração que, preferencialmente, utilizassem algoritmos de Aprendizado de Máquina e que utilizassem conjuntos de dados públicos para treinamento e teste dos modelos, para permitir reprodutibilidade. Para eliminar a necessidade de implementação foi verificado também se os autores disponibilizaram o código para treinamento e testes dos modelos, ou modelos pré-treinados.

Na Tabela~\ref{tab:revisao-bibliografica} há alguns dos artigos lidos e algumas informações sobre cada um.
